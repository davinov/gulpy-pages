Gulpy pages

A small recipe to start using beautiful languages like CoffeeScript, Jade and LESS and publishing it on GitHub pages.

Clone

git clone git@github.com:davinov/gulpy-pages.git [name-of-your-project]

Create your GitHub repo and change the original remote 

git remote set-url origin [your-github-repo-url]

Install npm / node then gulp 

[sudo] npm install -g gulp
npm install

Go to GitHub repo »

Develop

Modify files in app folder and compile then with 

gulp build

Or watch for changes in your code with 

gulp watch

Install dependencies using bower or directly add them in vendor folder

See your site live at localhost:3000 »

Publish

Compile, minify, commit your changes in public folder then push them on gh-pages branch 

./publish.sh

See your site published »
